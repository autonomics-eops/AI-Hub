403 Forbidden

Invalid or unsupported api-version in request.

Expected query parameter:

?api-version=2024-10-21

Correct request example (Chat Completions):

curl -X POST \
  "https://aihublz.jdecoffee.net/openai/deployments/<deployment-name>/chat/completions?api-version=2024-10-21" \
  -H "Content-Type: application/json" \
  -H "api-key: <your-key>" \
  -d '{
    "messages": [{"role": "user", "content": "Hello"}],
    "temperature": 0.7
  }'

Common mistakes:

  • Omitting the api-version parameter
  • Using a different or older version
  • Placing api-version in headers instead of the query string
